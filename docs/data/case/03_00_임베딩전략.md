```python
    model="text-embedding-3-large",
```

### 2.3 Matryoshka 차원 축소

**장점**:
- text-embedding-3-large의 높은 품질 유지
- 저장 공간 50% 절감 (3072 → 1536)
- 검색 속도 향상
- text-embedding-3-small(1536)과 동일한 크기

**설정**:

## ⚡ 4. 배치 처리 전략

### 4.1 API Rate Limits

**OpenAI API Tier 1 (기본)**:
- TPM (Tokens Per Minute): 200K
- RPM (Requests Per Minute): 500
- RPD (Requests Per Day): 10,000

### 4.2 배치 크기 설정

```python
BATCH_SIZE = 100  # 한 번에 처리할 청크 수
MAX_TOKENS_PER_BATCH = 150_000  # 배치당 최대 토큰 수
DELAY_BETWEEN_BATCHES = 2.0  # 초 단위
```

**근거**:
- 100개 청크 × 평균 1,500 tokens = 150K tokens/batch
- TPM 200K 제한 내에서 안전
- RPM 500 제한 여유 확보

### 4.4 배치 처리 로직

```python
def process_in_batches(chunks, batch_size=100):
    """청크를 배치로 나누어 처리"""
    total_batches = (len(chunks) + batch_size - 1) // batch_size

    for i in range(0, len(chunks), batch_size):
        batch = chunks[i:i+batch_size]
        batch_num = i // batch_size + 1

        print(f"Processing batch {batch_num}/{total_batches}...")

        # 배치 처리
        embeddings = create_embeddings_batch(batch)

        # 저장
        save_embeddings(batch, embeddings)

        # Rate limit 방지
        time.sleep(2.0)
```

## 🗄️ 7. PostgreSQL + pgvector 설정

### 7.1 데이터베이스 스키마

```sql
-- pgvector 확장 설치
CREATE EXTENSION IF NOT EXISTS vector;

-- 메인 테이블 생성
CREATE TABLE consumer_case_chunks (
    id SERIAL PRIMARY KEY,
    chunk_id VARCHAR(255) UNIQUE NOT NULL,
    data_source VARCHAR(50) NOT NULL,  -- 'crawling' or 'pdf'
    category VARCHAR(50) NOT NULL,      -- '상담', '해결', '조정'
    original_number VARCHAR(100) NOT NULL,
    field_name VARCHAR(100) NOT NULL,
    chunk_index INTEGER NOT NULL,
    total_chunks INTEGER NOT NULL,
    content TEXT NOT NULL,
    embedding vector(1536) NOT NULL,    -- 1536차원 벡터
    metadata JSONB,                     -- JSON 메타데이터
    created_at TIMESTAMP DEFAULT NOW()
);

-- 인덱스 생성
CREATE INDEX idx_chunk_id ON consumer_case_chunks(chunk_id);
CREATE INDEX idx_data_source ON consumer_case_chunks(data_source);
CREATE INDEX idx_category ON consumer_case_chunks(category);
CREATE INDEX idx_metadata ON consumer_case_chunks USING GIN(metadata);

-- 벡터 유사도 검색을 위한 HNSW 인덱스
CREATE INDEX ON consumer_case_chunks
USING hnsw (embedding vector_cosine_ops)
WITH (m = 16, ef_construction = 64);
```

### 7.2 HNSW 인덱스 파라미터

| 파라미터 | 값 | 설명 |
|---------|---|------|
| m | 16 | 그래프 연결 수 (기본값) |
| ef_construction | 64 | 구축 시 탐색 범위 |
| 인덱스 타입 | hnsw | Hierarchical Navigable Small World |
| 거리 함수 | cosine | 코사인 유사도 |

**대안: IVFFlat**
```sql
-- 더 빠른 구축, 약간 낮은 정확도
CREATE INDEX ON consumer_case_chunks
USING ivfflat (embedding vector_cosine_ops)
WITH (lists = 100);
```

### 7.3 데이터 삽입

```python
import psycopg2
from psycopg2.extras import execute_values

def insert_chunks_to_db(chunks_with_embeddings):
    """청크와 임베딩을 PostgreSQL에 삽입"""
    conn = psycopg2.connect(
        host="localhost",
        database="consumer_cases",
        user="postgres",
        password="your_password"
    )

    cursor = conn.cursor()

    # 데이터 준비
    data = [
        (
            chunk['chunk_id'],
            chunk['data_source'],
            chunk['category'],
            chunk['original_number'],
            chunk['field_name'],
            chunk['chunk_index'],
            chunk['total_chunks'],
            chunk['content'],
            chunk['embedding'],  # 리스트를 pgvector가 자동 변환
            json.dumps(chunk['metadata'])
        )
        for chunk in chunks_with_embeddings
    ]

    # 배치 삽입
    execute_values(
        cursor,
        """
        INSERT INTO consumer_case_chunks
        (chunk_id, data_source, category, original_number, field_name,
         chunk_index, total_chunks, content, embedding, metadata)
        VALUES %s
        ON CONFLICT (chunk_id) DO NOTHING
        """,
        data
    )

    conn.commit()
    cursor.close()
    conn.close()
```

### 7.4 유사도 검색 쿼리

```sql
-- 코사인 유사도로 상위 5개 검색
SELECT
    chunk_id,
    category,
    content,
    1 - (embedding <=> $1::vector) AS similarity
FROM consumer_case_chunks
WHERE category = '조정'  -- 옵션: 카테고리 필터
ORDER BY embedding <=> $1::vector
LIMIT 5;
```

```python
def validate_embedding(embedding):
    """임베딩 벡터 유효성 검사"""
    assert len(embedding) == 1536, "차원 오류"
    assert all(isinstance(x, float) for x in embedding), "타입 오류"
    assert not all(x == 0 for x in embedding), "제로 벡터 오류"

    # L2 노름 확인 (OpenAI 임베딩은 정규화됨)
    import math

    assert 0.99 <= norm <= 1.01, f"정규화 오류: {norm}"
```
