### 2.3 DB 연결 테스트

#### 방법 1: Python 스크립트로 테스트

`test_db_connection.py` 파일 생성:

```python
import psycopg2
from dotenv import load_dotenv
import os
from pathlib import Path

# .env 파일 로드
env_path = Path(__file__).parent / '.env'
load_dotenv(dotenv_path=env_path)

DB_CONFIG = {
    "host": os.getenv("DB_HOST"),
    "port": int(os.getenv("DB_PORT", "5432")),
    "database": os.getenv("DB_NAME"),
    "user": os.getenv("DB_USER"),
    "password": os.getenv("DB_PASSWORD")
}

try:
    print("🔌 AWS RDS 연결 테스트 중...")
    print(f"   Host: {DB_CONFIG['host']}")
    print(f"   Database: {DB_CONFIG['database']}")
    print(f"   User: {DB_CONFIG['user']}")

    conn = psycopg2.connect(**DB_CONFIG, connect_timeout=10)
    cursor = conn.cursor()

    # PostgreSQL 버전 확인
    cursor.execute("SELECT version();")
    version = cursor.fetchone()
    print(f"\n✅ 연결 성공!")
    print(f"   PostgreSQL Version: {version[0][:80]}...")

    # pgvector 확인
    cursor.execute("""
        SELECT extname, extversion
        FROM pg_extension
        WHERE extname = 'vector'
    """)
    pgvector = cursor.fetchone()
    if pgvector:
        print(f"   pgvector Version: {pgvector[1]}")

    # vector_chunks 테이블 확인
    cursor.execute("SELECT COUNT(*) FROM vector_chunks")
    chunk_count = cursor.fetchone()[0]
    print(f"   총 청크 수: {chunk_count:,}개")

    # 데이터셋별 통계
    cursor.execute("""
        SELECT dataset_type, COUNT(*)
        FROM vector_chunks
        GROUP BY dataset_type
    """)
    for row in cursor.fetchall():
        print(f"   - {row[0]}: {row[1]:,}개")

    cursor.close()
    conn.close()
    print("\n🎉 DB 연결 테스트 완료!")

except Exception as e:
    print(f"\n❌ 연결 실패: {e}")
    print("\n📌 체크리스트:")
    print("   1. .env 파일의 DB_PASSWORD가 정확한가요?")
    print("   2. 인터넷 연결이 정상인가요?")
    print("   3. 팀장에게 연결 정보를 확인하셨나요?")
```

### 4.3 방법 3: Python 스크립트로 커스텀 테스트

`my_test.py` 파일 생성:

```python
import requests
import json

API_URL = "http://localhost:8000"

def search_case(question, category=None, dataset=None):
    """사용자 질문으로 검색"""

    payload = {
        "query": question,
        "search_type": "hybrid",  # hybrid, vector, bm25 중 선택
        "top_k": 5
    }

    if category:
        payload["category_filter"] = category
    if dataset:
        payload["dataset_filter"] = dataset

    print(f"\n{'='*80}")
    print(f"질문: {question}")
    if category:
        print(f"카테고리 필터: {category}")
    if dataset:
        print(f"데이터셋 필터: {dataset}")
    print('='*80)

    response = requests.post(f"{API_URL}/search", json=payload)

    if response.status_code != 200:
        print(f"❌ 오류: {response.status_code}")
        print(response.text)
        return

    result = response.json()

    print(f"\n검색 결과: {result['total_results']}건")
    print(f"검색 시간: {result['search_time_ms']}ms\n")

    for i, item in enumerate(result['results'], 1):
        print(f"\n[{i}] 점수: {item['score']:.4f}")
        print(f"    데이터 타입: {item['dataset_type']}")

        if item.get('category'):
            print(f"    카테고리: {item['category']}")

        if item.get('law_name'):
            print(f"    법령명: {item['law_name']}")

        if item.get('source_url'):
            print(f"    출처 URL: {item['source_url'][:80]}...")

        # 내용 미리보기
        text_preview = item['text'][:200].replace('\n', ' ')
        print(f"\n    내용: {text_preview}...")
        print(f"    {'-'*76}")

# 테스트 질문들
if __name__ == "__main__":
    # 예시 1: 환불 관련 질문
    search_case("노트북 디자인이 별로라서 환불받고 싶은데, 유사한 사례가 있어?")

    input("\n다음 검색을 보려면 Enter를 누르세요...")

    # 예시 2: 배송 지연 문제 (해결사례만)
    search_case("배송이 한 달째 안 오는데 어떻게 해야 하나요?", category="해결")

    input("\n다음 검색을 보려면 Enter를 누르세요...")

    # 예시 3: 법령 검색
    search_case("소비자기본법 제16조 내용이 궁금해요", dataset="law_guide")

    print("\n테스트 완료!")
```


**해결 방법:**

1. **.env 파일 확인**
   ```bash
   # DB_HOST가 정확한지 확인
   DB_HOST=your-db-instance.ap-northeast-2.rds.amazonaws.com
   DB_PORT=5432
   DB_NAME=ddoksori
   DB_USER=postgres
   DB_PASSWORD=올바른비밀번호
   ```

2. **인터넷 연결 확인**
   ```bash
   # Windows
   ping your-db-instance.ap-northeast-2.rds.amazonaws.com

   # 연결 테스트
   Test-NetConnection -ComputerName your-db-instance.ap-northeast-2.rds.amazonaws.com -Port 5432
   ```

3. **팀장에게 확인**
   - AWS RDS 인스턴스가 실행 중인지

## 9. API 서버 접속 문제 해결

### 9.1 문제 상황

`python 03_01_search_api.py` 실행 후 `http://localhost:8000/docs` 접속이 안 되는 경우

### 9.2 서버 실행 확인

#### 1단계: 터미널 출력 확인

서버가 정상적으로 실행되면 다음과 같은 메시지가 출력됩니다:

```
INFO:     Started server process [12345]
INFO:     Waiting for application startup.
INFO:     Application startup complete.
INFO:     Uvicorn running on http://127.0.0.1:8000 (Press CTRL+C to quit)
```

#### 2단계: 포트 사용 확인

**Windows:**
```bash
netstat -ano | findstr :8000
```

**Mac/Linux:**
```bash
lsof -i :8000
```

포트가 사용 중이면 프로세스 번호(PID)가 표시됩니다.

### 9.3 일반적인 오류 및 해결 방법

#### 오류 1: DB 연결 실패

**증상:**
```
sqlalchemy.exc.OperationalError: could not connect to server
```

**원인:**
- `.env` 파일 누락 또는 잘못된 비밀번호
- AWS RDS 보안 그룹 설정 문제
- 네트워크 연결 문제

**해결:**
1. `.env` 파일이 `DB/` 폴더에 있는지 확인
2. `DB_PASSWORD` 값이 정확한지 확인 (팀장에게 문의)
3. 인터넷 연결 확인
4. VPN 사용 중이면 해제 후 재시도

#### 오류 2: 포트가 이미 사용 중

**증상:**
```
ERROR: [Errno 10048] error while attempting to bind on address ('0.0.0.0', 8000)
```

**원인:**
- 다른 프로그램이 8000번 포트 사용 중
- 이전 실행이 제대로 종료되지 않음

**해결:**

**Windows:**
```bash
# 포트 사용 프로세스 찾기
netstat -ano | findstr :8000

# 프로세스 종료 (PID는 위에서 확인한 번호)
taskkill /PID [프로세스번호] /F
```

**Mac/Linux:**
```bash
# 포트 사용 프로세스 찾기
lsof -i :8000

# 프로세스 종료
kill -9 [PID]
```

또는 다른 포트로 실행:
```bash
uvicorn 03_01_search_api:app --host 0.0.0.0 --port 8001
# 그 다음 http://localhost:8001/docs 접속
```

#### 오류 3: 필요한 패키지 누락

**증상:**
```
ModuleNotFoundError: No module named 'fastapi'
```

**해결:**
```bash
pip install fastapi uvicorn sqlalchemy psycopg2-binary openai python-dotenv
```

#### 오류 4: 방화벽 차단

**증상:**
- 서버는 실행되지만 브라우저에서 접속 안 됨

**해결:**

**Windows:**
1. Windows Defender 방화벽 → 고급 설정
2. 인바운드 규칙 → 새 규칙
3. 포트 → TCP → 특정 로컬 포트 → 8000
4. 연결 허용 → 완료

**Mac:**
```bash
# 방화벽 상태 확인
sudo /usr/libexec/ApplicationFirewall/socketfilterfw --getglobalstate

# 파이썬 허용
sudo /usr/libexec/ApplicationFirewall/socketfilterfw --add /usr/bin/python3
```
